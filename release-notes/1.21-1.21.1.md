# KoHs Crystal Tweaks 1.0.0 — Minecraft 1.21–1.21.1

Beta/pre-release distribution of the preserved Fabric build for Minecraft `>=1.21 <1.21.2` and Java 21.

## Technical details

- Yarn mappings: `1.21+build.9`.
- Fabric Loader: `0.18.6`.
- Fabric API: `0.102.0+1.21`.
- Local crystal prediction and seamless server-entity reconciliation.
- Client-side crystal optimizer without automated attacks or additional combat packets.
- Vanilla-pipeline explosion sound replacement with WAV, OGG, and MP3 import support.
- Configuration screen protected from unwanted vanilla blur.

This legacy artifact keeps its original internal version, `1.0.0+mc1.21`. Placement Fix is not included; it starts with Minecraft 1.21.10.

Preserved build: `kohs-crystal-tweaks-1.0.0+mc1.21.jar`. A Minecraft client was not launched during this publication pass.
