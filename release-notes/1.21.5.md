# KoHs Crystal Tweaks 1.0.0 — Minecraft 1.21.5

Beta/pre-release distribution of the preserved Fabric build for Minecraft `>=1.21.5 <1.21.6` and Java 21.

## Technical details

- Yarn mappings: `1.21.5+build.1`.
- Fabric Loader: `0.17.2`.
- Fabric API: `0.128.2+1.21.5`.
- Dedicated `ModelPart` integration for the 1.21.5 renderer API.
- `SafeCrystalMixin` avoids the binary-fragile call responsible for a reported 1.21.5 crash.
- Local crystal prediction, seamless reconciliation, client-side visual optimization, and vanilla-pipeline explosion sound replacement.

This legacy artifact keeps its original internal version, `1.0.0+mc1.21.5`. Placement Fix is not included; it starts with Minecraft 1.21.10.

Preserved build: `kohs-crystal-tweaks-1.0.0+mc1.21.5.jar`. A Minecraft client was not launched during this publication pass.
