# KoHs Crystal Tweaks 1.1.0-beta.3 — Minecraft 1.21.11

Fabric beta for Java 21, Yarn `1.21.11+build.5`, Fabric Loader `0.17.2`, and Fabric API `0.140.0+1.21.11`.

## Technical changes

- Added `Rapid Attack Fix` to `Tweaks`, enabled by default.
- Fixed the attack race caused by very fast left/right butterfly clicks reaching a predicted crystal before its real server entity is loaded.
- Intercepts only after vanilla has accepted the physical attack and reached `ClientPlayerInteractionManager.attackEntity`.
- Stores one deduplicated pending intent on the local prediction; it cannot generate repeated attack packets.
- Waits for the matching real End Crystal and its server-assigned entity ID before sending the attack through the normal interaction manager.
- Expires the pending intent with the existing local-prediction timeout when no matching server entity arrives.
- Keeps Placement Fix, accepted-result prediction, vanilla base validation, adaptive timeout, per-part tint, and responsive layout corrections from beta.2.
- Replaced fixed description blocks with hover tooltips for every configurable feature and slider.

Rapid Attack Fix does not synthesize clicks, guess entity IDs, automate targets, or retry attacks. It preserves at most one vanilla-validated attack during the local-to-server crystal handoff.

Verified with `gradlew.bat clean build --no-daemon`. Minecraft was not launched.
