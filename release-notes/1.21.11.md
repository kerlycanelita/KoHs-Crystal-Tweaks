# KoHs Crystal Tweaks 1.1.0-beta.1 — Minecraft 1.21.11

Fabric beta for Java 21, Yarn `1.21.11+build.5`, Fabric Loader `0.17.2`, and Fabric API `0.140.0+1.21.11`.

## Technical changes

- Integrated Placement Fix into `ClientPlayerInteractionManager.interactBlock` by retargeting the current `BlockHitResult`.
- Added an enabled-by-default toggle to `Tweaks`; `Aceptar` (Accept) disables it and `Restablecer` (Restore) keeps it enabled.
- Local prediction is created only after `ActionResult.isAccepted()`.
- Initial prediction timeout now matches the configured 12 ticks.
- Removed invalid crying obsidian prediction; valid bases are obsidian and bedrock, matching vanilla.
- Applied separate frame/core tint during actual per-part rendering without duplicating queued geometry.
- Made the configuration screen responsive at compact logical sizes and high GUI scales.

Placement Fix reuses one vanilla interaction and creates no extra packet, repeated click, item switch, or remote target selection.

Verified with `gradlew.bat clean build --no-daemon`. Minecraft was not launched.
