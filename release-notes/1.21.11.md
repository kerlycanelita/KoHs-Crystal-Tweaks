# KoHs Crystal Tweaks 2.0.0-beta.3 — Minecraft 1.21.11

Fabric release 2 beta for Java 21, Yarn `1.21.11+build.5`, Fabric Loader `0.17.2`, and Fabric API `0.140.0+1.21.11`.

## Safe Crystal scope correction

- Safe Crystal remains OFF by default in the `Tweaks` tab.
- When enabled, it now protects normal obsidian only while an End Crystal is held.
- Crying obsidian was removed from both `attackBlock` and `updateBlockBreakingProgress` guards and follows untouched vanilla behavior.
- The control still switches immediately with no `Accept` / `Restore` warning.
- Disabling Safe Crystal still returns before player, world, held-item, or block-state lookup and never cancels vanilla block breaking.

## Default option state

- Optimization ON: `Local Crystal`, `Seamless Mode`.
- Tweaks ON: `Placement Fix`, `Rapid Attack Fix`, `Crystal Flotation`.
- Tweaks OFF: `Safe Crystal`, `Static Crystal`.
- `Spin Speed` starts at 100%.

## Verification

- Clean Gradle build and all six automated tests completed successfully.
- Minecraft was not launched.
