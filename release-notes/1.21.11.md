# KoHs Crystal Tweaks 2.0.0-beta.2 — Minecraft 1.21.11

Fabric release 2 beta for Java 21, Yarn `1.21.11+build.5`, Fabric Loader `0.17.2`, and Fabric API `0.140.0+1.21.11`.

## Safe Crystal control

- Adds a persistent `Safe Crystal` ON/OFF switch to the `Tweaks` tab.
- Safe Crystal is OFF by default, including automatic migration of older configuration files that do not contain `safeCrystalEnabled`.
- An explicit user selection remains persistent after saving.
- Switches immediately without an `Accept` / `Restore` confirmation.
- When enabled, prevents `attackBlock` and `updateBlockBreakingProgress` from damaging obsidian or crying obsidian while an End Crystal is held.
- When disabled, each callback returns immediately after one cached configuration read, before player, world, held-item, or block-state lookup; vanilla block-breaking behavior is not cancelled.
- Does not alter crystal placement through `interactBlock`, crystal entity attacks through `attackEntity`, cooldowns, clicks, or packets.

## Responsive Tweaks layout

- Keeps the existing single-column layout at normal logical sizes.
- Uses two columns on compact screens, fitting Placement Fix, Rapid Attack Fix, Safe Crystal, Static Crystal, Crystal Flotation, and Spin Speed above the Close button.
- Keeps all descriptions in hover tooltips and adds no warning to Safe Crystal.

## Verification

- Clean Gradle build and all six automated tests completed successfully.
- The migration test verifies `safeCrystalEnabled=false` when an older JSON file lacks the field.
- Minecraft was not launched.
