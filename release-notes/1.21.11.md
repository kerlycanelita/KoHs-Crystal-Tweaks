# KoHs Crystal Tweaks 1.1.0-beta.6 — Minecraft 1.21.11

Fabric beta for Java 21, Yarn `1.21.11+build.5`, Fabric Loader `0.17.2`, and Fabric API `0.140.0+1.21.11`.

## Incompatibility guard

- Detects unsafe optimizer combinations before KoHs gameplay mixins are applied.
- Explicitly blocks Marlow's Crystal Optimizer (`marlowcrystal`). A confirmed beta.5 log showed both mods registering `marlowcrystal:opt_out`; Fabric rejected the duplicate payload and crashed during client entrypoint startup.
- Stops KoHs before it registers prediction, placement, sound, gameplay events, or the mirrored Marlow compatibility payloads.
- Scans unknown Fabric mixin metadata and class annotations without loading candidate classes.
- Blocks an unknown mod only if it directly targets an internal KoHs class, or if a crystal-related mixin targets the same class and exact critical method as KoHs.
- Does not block a mod merely for sharing a Minecraft target class; different methods and unrelated mods remain allowed.

## Mandatory diagnostic screen

- Replaces the normal Mod Menu screen while a conflict exists.
- Displays the conflicting mod name, ID, version, technical category, and detected class/method points.
- Explains every consequence in English first and Spanish second.
- Uses bounded margins and a scrollable report so it remains usable at high GUI scales and compact logical resolutions.
- Cannot be dismissed with Escape or replaced by another screen; the only action is `Close Minecraft / Cerrar Minecraft`, which schedules a clean client shutdown.

## Safety and verification

- KoHs mixins are disabled by `IMixinConfigPlugin.shouldApplyMixin` before application; gameplay services are independently disabled by both main and client entrypoint guards.
- Generic scanning fails open on malformed third-party metadata to avoid turning an inspection error into a false incompatibility.
- Automated tests cover the real KoHs critical hook bytecode, selector normalization, exact method overlap, different-method tolerance, and unrelated-mod tolerance.
- Existing legitimate input guarantees remain unchanged: no generated clicks, cooldown removal, guessed entity IDs, automatic retries, or extra attack/use packets.

Verified with `gradlew.bat clean build --no-daemon`. Minecraft was not launched.
