# KoHs Crystal Tweaks 2.0.1 — Minecraft 26.1.2

Official-name Java 25 build using Fabric Loader `0.19.3` and Fabric API `0.154.2+26.1.2`.

## Installation

Use `kohs-crystal-tweaks-2.0.1+mc26.1.2.jar` only. This artifact declares Minecraft 26.1.2 exactly and contains official-name 26.x classes; the separate 1.21.11 artifact is not binary-compatible with 26.x.

## Fixes and behavior

- Ports the complete release 2 placement, ordered-input, rapid-attack, safety, UI, rendering, sound, networking, and incompatibility behavior.
- Uses `MultiPlayerGameMode` and waits for a real `EndCrystal` entity ID before sending a preserved predicted attack.
- Leaves ordinary single clicks on Minecraft's direct input path and reserves ordered replay for order-sensitive same-tick sequences.
- Keeps real-crystal cleanup immediate when optional Local Crystal prediction is OFF.
- Restores runtime WAV/OGG/MP3 explosion replacement through `SoundBufferLibrary`, `SoundBuffer`, and `SoundEngine`.
- Applies frame/core tint, spin, flotation, static mode, and seamless handoff through the submit renderer.
- Keeps fresh defaults and bilingual warning scope identical to the stable 1.21.11 build.

No synthesized action, guessed ID, retry loop, cooldown change, or extra interaction packet is used. The clean build and all 10 tests passed; Minecraft was not launched.
