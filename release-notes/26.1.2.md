# KoHs Crystal Tweaks 1.1.0-beta.1 — Minecraft 26.1.2

Beta Fabric para Java 25, mappings oficiales de Mojang, Fabric Loader `0.19.3` y Fabric API `0.153.0+26.1.2`.

## Cambios técnicos

- Placement Fix integrado en `MultiPlayerGameMode.useItemOn` sin construir paquetes adicionales.
- Toggle activado por defecto en `Tweaks`, con advertencia y acciones `Aceptar` / `Restablecer`.
- Predicción local disparada sólo para `InteractionResult.Success`.
- Timeout inicial de predicción alineado con los 12 ticks configurados.
- Port completo del tint separado frame/core al renderer por cola de 26.1.2.
- Port de `Spin Speed`, `Crystal Flotation` y `Static Crystal`, incluido el beam.

Build verificado con JDK 25 y `gradlew.bat build --no-daemon`. Minecraft no fue ejecutado.
