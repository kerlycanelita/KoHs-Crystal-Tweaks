# KoHs Crystal Tweaks 1.1.0-beta.2 — Minecraft 26.1.2

Fabric beta for Java 25, official Mojang mappings, Fabric Loader `0.19.3`, and Fabric API `0.153.0+26.1.2`.

## Technical changes

- Converted the Placement Fix warning, description, and buttons to English.
- Embedded Fabric/Mod Menu metadata declares the MIT license.
- Integrated Placement Fix into `MultiPlayerGameMode.useItemOn` without constructing additional packets.
- Added an enabled-by-default toggle to `Tweaks`; `Accept` disables it and `Restore` keeps it enabled.
- Local prediction runs only after `InteractionResult.Success`.
- Initial prediction timeout now matches the configured 12 ticks.
- Ported separate frame/core tint to the 26.1.2 submit-based renderer.
- Ported `Spin Speed`, `Crystal Flotation`, and `Static Crystal`, including beam positioning.
- Kept the configuration UI within current logical bounds at compact resolutions and high GUI scales.

Placement Fix reuses one vanilla interaction and creates no extra packet, repeated click, item switch, or remote target selection.

Verified with JDK 25 and `gradlew.bat clean build --no-daemon`. Minecraft was not launched. Runtime custom sound replacement remains incomplete on this branch.
