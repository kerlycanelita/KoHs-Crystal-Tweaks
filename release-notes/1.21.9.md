# KoHs Crystal Tweaks 1.0.0 — Minecraft 1.21.9

Beta/pre-release distribution of the preserved Fabric build for Minecraft `>=1.21.9 <1.21.10` and Java 21.

## Technical details

- Yarn mappings: `1.21.9+build.1`.
- Fabric Loader: `0.17.2`.
- Fabric API: `0.134.1+1.21.9`.
- First isolated branch using the modern queued entity renderer.
- Seamless integration targets `OrderedRenderCommandQueue` and `EndCrystalEntityRenderState`.
- Modern Marlow version/opt-out/acknowledgement identifiers with per-server state.
- The optimizer reacts only to the vanilla attack already issued by the player and performs local visual cleanup.

This artifact keeps its original internal version, `1.0.0+mc1.21.9`. Placement Fix is not included; it starts with Minecraft 1.21.10.

Preserved build: `kohs-crystal-tweaks-1.0.0+mc1.21.9.jar`. A Minecraft client was not launched during this publication pass.
