# KoHs Crystal Tweaks 1.0.0 — Minecraft 1.21.2–1.21.4

Beta/pre-release distribution of the preserved Fabric build for Minecraft `>=1.21.2 <1.21.5` and Java 21.

## Technical details

- Yarn mappings: `1.21.2+build.1`.
- Fabric Loader: `0.17.2`.
- Fabric API: `0.106.1+1.21.2`.
- Uses the renderer API compatible across 1.21.2–1.21.4.
- Local crystal prediction, seamless reconciliation, and client-side visual optimization.
- Vanilla-pipeline explosion sound replacement.
- Compatibility upper bound prevents loading against the incompatible 1.21.5 internals.

This legacy artifact keeps its original internal version, `1.0.0+mc1.21.2`. Placement Fix is not included; it starts with Minecraft 1.21.10.

Preserved build: `kohs-crystal-tweaks-1.0.0+mc1.21.2.jar`. A Minecraft client was not launched during this publication pass.
