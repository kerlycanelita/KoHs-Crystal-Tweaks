# KoHs Crystal Tweaks 1.1.0-beta.1 — Minecraft 1.21.10

Beta Fabric para Java 21, Yarn `1.21.10+build.3`, Fabric Loader `0.17.2` y Fabric API `0.138.4+1.21.10`.

## Cambios técnicos

- Placement Fix integrado en `ClientPlayerInteractionManager.interactBlock` mediante retarget del `BlockHitResult` actual.
- Toggle activado por defecto en `Tweaks`, con advertencia y acciones `Aceptar` / `Restablecer`.
- La predicción local sólo se crea si `ActionResult.isAccepted()`.
- Timeout inicial de predicción alineado con los 12 ticks configurados.
- Eliminada la base inválida crying obsidian; se usan obsidiana y bedrock como vanilla.
- Tinte separado de frame/core aplicado al render real por pieza, sin duplicar geometría encolada.
- Config screen responsivo para tamaños lógicos compactos.

Build verificado con `gradlew.bat build --no-daemon`. Minecraft no fue ejecutado.
