# KoHs Crystal Tweaks 1.0.0 — Minecraft 1.21.6–1.21.8

Beta/pre-release distribution of the preserved Fabric build for Minecraft `>=1.21.6 <1.21.9` and Java 21.

## Technical details

- Yarn mappings: `1.21.6+build.1`.
- Fabric Loader: `0.19.2`.
- Fabric API: `0.128.2+1.21.6`.
- Dedicated compatibility group for the intermediate 1.21.6+ client API.
- `SafeCrystalMixin` avoids a binary-fragile player-world accessor.
- Local crystal prediction, seamless reconciliation, client-side visual optimization, and vanilla-pipeline explosion sound replacement.

This legacy artifact keeps its original internal version, `1.0.0+mc1.21.6`. Placement Fix is not included; it starts with Minecraft 1.21.10.

Preserved build: `kohs-crystal-tweaks-1.0.0+mc1.21.6.jar`. A Minecraft client was not launched during this publication pass.
